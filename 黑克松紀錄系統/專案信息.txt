========================================
    黑克松紀錄系統 v1.0.0
========================================

📅 創建日期：2025-01-20
🔧 技術棧：Node.js + Express + HTML/CSS/JS
📊 資料格式：JSON
🌐 服務端口：3000

🚀 核心功能：
• 完整的 CRUD 操作
• 即時資料保存
• 自動備份系統
• 資料匯入匯出
• 響應式設計

📁 專案結構：
黑克松紀錄系統/
├── 黑克松紀錄.html          # 前端主頁面
├── 黑克松資料.jason          # 資料存儲文件
├── server.js               # Express 後端服務器
├── package.json            # Node.js 專案配置
├── start.bat              # Windows 啟動腳本
├── start.sh               # Mac/Linux 啟動腳本
├── README.md              # 詳細使用說明
├── 快速啟動.md             # 快速啟動指南
└── 專案信息.txt            # 此文件

💡 使用建議：
1. 首次使用請運行啟動腳本
2. 所有資料會自動保存到 JSON 文件
3. 支援備份和恢復功能
4. 可離線使用（localStorage 備用）

🔒 安全提醒：
• 此系統設計用於本地開發
• 不建議直接暴露到公網
• 資料完全由您掌控
